# Arizona Environmental Health Atlas

**Live:** https://jgyoon00.github.io/map/

Interactive map of tract- and county-level environmental health indicators across Arizona. Built with Leaflet.js using public data from EPA EJScreen, CDC BRFSS, PRISM, WFIGS, and Census TIGER/Line.

## Tabs
- **Air Quality** — PM2.5, ozone (tract-level, EJScreen 2024)
- **Climate** — Temperature, precipitation, NDVI (county-level, PRISM)
- **Equity** — Low income, people of color (tract-level, ACS)
- **Health** — Obesity, diabetes, COPD, smoking, inactivity, cancer (county-level, CDC BRFSS / AZ Cancer Registry)
- **Wildfire** — Fire perimeters >500 acres, county fire stats (WFIGS/NIFC)
